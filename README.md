# Vectorless RAG Benchmark

Benchmarking embedding-free retrieval paradigms for production RAG systems.

## Overview
This project evaluates 5 retrieval strategies that do **not** require vector databases:
- BM25 (sparse keyword)
- TF-IDF
- LLM-as-reranker
- ColBERT late-interaction
- Hybrid BM25 + LLM judge
